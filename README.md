🛒 E-Commerce Flutter App
A modern and responsive E-Commerce Mobile Application built using Flutter.
This project is currently in development and focuses on clean UI, smooth onboarding experience, and scalable architecture.

✨ Beautiful Onboarding Screens

⚙️ Installation

Clone the repository:

git clone https://github.com/Subhan-6/E_Commerce-App.git

Navigate to project folder:

cd E_Commerce-App

Install dependencies:

flutter pub get

Run the app:

flutter run
